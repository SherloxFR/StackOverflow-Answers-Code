# Winston Log JSON

To run the project:
```bash
git clone https://github.com/SherloxFR/StackOverflow-Answers-Code.git
cd StackOverflow-Answers-Code/winston-log-json
npm install
npm start
```

The combined log file is located at `logs/combined.log`.
